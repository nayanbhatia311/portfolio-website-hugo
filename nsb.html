Network Simulation Bridge (NSB)

Open-source, platform-agnostic co-simulation infrastructure for connecting real or emulated applications to network simulation backends.

Overview

NSB lets researchers, developers, and students connect a front-end program to a back-end network simulator. The public i-NRG project page describes NSB as a core service for message management plus front-end and back-end APIs for simulator integration.

The project is designed to be open-source and agnostic to the front-end interface, back-end simulator, and deployment environment. Current development focuses on simulator support, including OMNeT++ and ns-3, along with protocol standardization, performance optimization, and higher-level APIs.

IEEE ICDCS 2025 Tutorial

The ICDCS 2025 tutorial page describes NSB as an open-source and platform-agnostic network co-simulation tool for integrating network simulations into distributed applications and systems. The tutorial covered NSB design, implementation, and usage with OMNeT++ and ns-3.

Tutorial: Network Simulation Bridge: Bridging Distributed Applications with Network Simulators

NSF Funding

NSB is supported by NSF POSE Award 2449377, Open-Sourcing the Network Simulation Bridge for Networked Applications Development. The award supports building an open-source ecosystem around NSB for realistic, reproducible testing of distributed applications.
